## What is Trellis?

Trellis is a framework that gives AI coding assistants persistent knowledge about your project — coding conventions, task context, and session history.

Think of it like onboarding docs for your AI. New engineers need context to write good code. So does AI.

## What you can do

- **Enforce coding standards** — AI follows your patterns, not generic ones
- **Track task context** — Each feature gets its own relevant files
- **Continue where you left off** — Session journals preserve history

## How it works

1. **Specs** — Document your coding conventions in `.trellis/spec/`. These get injected into AI context automatically.

2. **Tasks** — Track work in `.trellis/tasks/`. Each task can have its own context files that tell the AI what's relevant.

3. **Workspace** — Session journals in `.trellis/workspace/` let you pick up where you left off.

## Who is this for?

Trellis works best if you:

- Use Claude Code or Cursor for daily development
- Work on a codebase with specific conventions
- Want AI to follow your team's patterns, not generic ones
