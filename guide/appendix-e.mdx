## Appendix E: `worktree.yaml` (removed)

This appendix previously documented `.trellis/worktree.yaml`, the configuration file for the Multi-Agent Pipeline. Both the pipeline and this file were removed in **0.5.0-beta.0** along with the `/parallel` command, the `dispatch` agent, and Ralph Loop.

If you're looking for the old content because you maintain a pre-0.5 project, the previous Appendix E is preserved in the Release-track documentation (which still targets 0.4.0).

If you were using the old `verify:` field to enforce lint / typecheck / test on AI output, the replacement is:

- Put the same commands in your package spec (e.g. `.trellis/spec/backend/quality.md`).
- The `trellis-check` sub-agent reads that spec and runs the same checks on every diff, with its own self-fix loop.

No Trellis config file is involved.
