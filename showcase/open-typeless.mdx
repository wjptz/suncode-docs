# About

A macOS voice input app, built with Trellis in 1 day.

[![open-typeless](https://opengraph.githubassets.com/1/mindfold-ai/open-typeless)](https://github.com/mindfold-ai/open-typeless)

## How it was built

Copied specs from an existing Electron project, AI filtered and organized them into 3 task batches. Current Trellis runs this style of work with native Git worktrees plus one session-scoped Trellis task per AI window.
