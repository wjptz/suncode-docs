# About

A macOS voice input app, built with Trellis in 1 day.

[![open-typeless](https://opengraph.githubassets.com/1/mindfold-ai/open-typeless)](https://github.com/mindfold-ai/open-typeless)

## How it was built

Copied specs from an existing Electron project, AI filtered and organized them into 3 task batches. Used `/trellis:parallel` to launch 3 worktree agents for parallel development, PRs created automatically upon completion.
