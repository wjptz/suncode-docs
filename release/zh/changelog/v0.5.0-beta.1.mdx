**0.5 首个正式 beta 发布。** 代码和 `0.5.0-beta.0` 开发版完全一致。`beta.0` → `beta.1` 的 bump 是 `pnpm release:beta` 自动做的（通过 `pnpm version prerelease --preid beta`），所以 `beta.0` 从未发到 npm —— `beta.1` 才是 skill-first 架构的第一个正式发布版本。

所有重活（5 个 skill 的 command→skill 迁移、138 条清理老命令 + iFlow + multi-agent + Ralph Loop 的 safe-file-delete、breaking-change `--migrate` gate、confirm prompt 里的 per-entry `reason` 字段、worktree backup 排除、build `clean` 步骤）都在 `0.5.0-beta.0` manifest 里定义，从 0.4.x 升级时都会触发。

完整迁移说明见 [`v0.5.0-beta.0` changelog](/zh/changelog/v0.5.0-beta.0)。

## 迁移

从 0.4.x 升级：跑 `trellis update --migrate`。breaking-change gate 要求显式加这个 flag —— 不加的话 `trellis update` 会 exit 1 并提示你加上。

安装：`npm install -g @mindfoldhq/trellis@beta`
