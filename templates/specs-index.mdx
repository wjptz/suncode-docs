Spec templates help you quickly set up coding guidelines for your project. Copy, customize, use.

## Available Templates

| Template                                           | Stack      | Description             |
| -------------------------------------------------- | ---------- | ----------------------- |
| [Node.js + Express](/templates/specs-node)         | Backend    | Express with TypeScript |
| [Python + FastAPI](/templates/specs-python)        | Backend    | FastAPI with SQLAlchemy |
| [React + TypeScript](/templates/specs-react)       | Frontend   | React with Zustand      |
| [Next.js + tRPC + Prisma](/templates/specs-nextjs) | Full-stack | Next.js app router      |

## How to Use

1. Pick a template that matches your tech stack
2. Copy to `.trellis/spec/backend/` or `.trellis/spec/frontend/`
3. Customize for your project's specific conventions
4. Remove sections that don't apply
