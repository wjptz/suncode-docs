Spec templates help you quickly set up coding guidelines for your project. Copy, customize, and use.

## Available Templates

| Template                          | Stack | Description                  |
| --------------------------------- | ----- | ---------------------------- |
| [General Specs](/templates/specs) | Any   | Universal coding conventions |

## How to Use

1. Pick a template that matches your stack
2. Copy to `.trellis/spec/`
3. Edit for your project's specific conventions
4. Delete sections that don't apply
