Command templates provide ready-to-use slash commands for common development workflows.

## Available Templates

| Template                                | Purpose   | Description                 |
| --------------------------------------- | --------- | --------------------------- |
| [General Commands](/templates/commands) | Workflows | Common development commands |

## How to Use

1. Copy the command file to `.claude/commands/your-project/`
2. Rename if needed
3. Test with `/your-project:command-name`
