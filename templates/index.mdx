## How to use templates

### Specs

1. Pick a template that matches your stack
2. Copy to `.trellis/spec/`
3. Edit for your project's specific conventions
4. Delete sections that don't apply

### Commands

1. Copy the command file to `.claude/commands/your-project/`
2. Rename if needed
3. Test with `/your-project:command-name`

### Skills

1. Copy the skill folder to `.claude/skills/`
2. Skills auto-load when relevant
3. Some skills need configuration - check the README

## Contributing templates

Made something useful? Share it.

1. Fork the Trellis repo
2. Add your template to `templates/`
3. Include a README
4. Open a PR

Good templates are specific, well-documented, and tested on real code.
