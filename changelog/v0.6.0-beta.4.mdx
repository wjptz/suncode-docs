## Bug Fixes

- **`better-sqlite3` dependency removed.** Reverts the OpenCode SQLite reader added in 0.6.0-beta.3. Fixes `npm install -g @mindfoldhq/trellis@beta` failure when the prebuilt binary download fails and no local C toolchain is available.
- **OpenCode platform degraded.** `tl mem list / search / extract` on platform `opencode` returns empty + a one-shot stderr warning. Claude and Codex paths unchanged.

## Upgrade

```bash
npm install -g @mindfoldhq/trellis@beta
trellis update
```
