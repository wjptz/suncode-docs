---
title: 'trellis-meta'
description: '定制 Trellis 的必备技能'
---

官方元技能，用于理解和定制 Trellis。安装后，AI 可以帮你：

- 为你的工作流加专门的 agent
- 改上下文注入的方式
- 加项目特定的命令
- 修改质量检查

## 安装

<Tabs>
  <Tab title="npx skills（推荐）">
    支持 Claude Code、Cursor、OpenCode、iFlow、Codex、Kilo、Kiro、Gemini CLI、Antigravity 等更多 agents。

    ```bash
    npx skills add mindfold-ai/docs --skill trellis-meta
    ```

    或安装所有可用 skills：

    ```bash
    npx skills add mindfold-ai/docs
    ```

    选项：

    | 参数 | 说明 |
    | --- | --- |
    | `-g` | 全局安装（`~/.claude/skills/`） |
    | `-a claude-code` | 指定目标 agent |
    | `-y` | 非交互模式 |

  </Tab>
  <Tab title="Claude Code Plugin">
    **第一步**：注册 skill 来源

    ```bash
    /plugin marketplace add mindfold-ai/docs
    ```

    **第二步**：安装 skill

    ```bash
    /plugin install trellis-meta
    ```

    安装后，skill 显示为 `trellis-marketplace:trellis-meta`。

  </Tab>
</Tabs>

## 验证安装

检查 skill 是否可用：

```
你有哪些 skills？
```

Claude 应该在回复中列出 `trellis-meta` 或 `trellis-marketplace:trellis-meta`。

## 使用

安装后，直接告诉 AI 你想做什么：

```
我想加一个 deploy agent 处理部署流程
```

```
帮我修改 check hook 加一个自定义验证命令
```

```
我要加一个新的工作流阶段叫 review
```

AI 会自动使用技能里的文档，给你正确的修改步骤。

## 包含内容

| 目录             | 内容                                                             |
| ---------------- | ---------------------------------------------------------------- |
| `core/`          | 核心系统（workspace, tasks, specs, scripts）                     |
| `claude-code/`   | Claude Code 专属功能（hooks, agents, Ralph Loop, multi-session） |
| `how-to-modify/` | 修改指南（加 agent, 加命令, 改 hook 等）                         |
| `meta/`          | 平台兼容性、自迭代指南                                           |
