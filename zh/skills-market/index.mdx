技能为 Trellis 扩展专业知识和工作流。当前 Trellis 安装会自动包含内置 Trellis skills，marketplace skills 继续作为兼容路径和专业领域扩展。

通过 [skills.sh](https://skills.sh) 一键安装 marketplace skills：

```bash
npx skills add mindfold-ai/marketplace
```

## 官方技能

| 技能                                                                                 | 描述                                                         | 安装                                                                        |
| ------------------------------------------------------------------------------------ | ------------------------------------------------------------ | --------------------------------------------------------------------------- |
| [trellis-meta](/zh/skills-market/trellis-meta)                                       | 定制和修改 Trellis                                           | `trellis init` 内置；marketplace 安装用于兼容                               |
| [trellis-spec-bootstarp](/zh/skills-market/trellis-spec-bootstarp)                   | 从真实代码库引导生成项目专属 Trellis specs                   | 随 Trellis 内置安装                                                         |
| [frontend-fullchain-optimization](/zh/skills-market/frontend-fullchain-optimization) | 用 Web Vitals 诊断并优化前端性能                             | `npx skills add mindfold-ai/marketplace -s frontend-fullchain-optimization` |
| [mem-recall](/zh/skills-market/mem-recall)                                           | 通过 `trellis mem` 跨 Claude / Codex / OpenCode 回看历史对话 | `npx skills add mindfold-ai/marketplace -s mem-recall`                      |
