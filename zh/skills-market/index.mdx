技能为 Trellis 扩展专业知识和工作流。安装一个技能，AI 就能立即帮你处理该领域的问题。

通过 [skills.sh](https://skills.sh) 一键安装：

```bash
npx skills add mindfold-ai/Trellis/marketplace
```

## 官方技能

| 技能                                           | 描述               | 安装                                              |
| ---------------------------------------------- | ------------------ | ------------------------------------------------- |
| [trellis-meta](/zh/skills-market/trellis-meta) | 定制和修改 Trellis | `npx skills add mindfold-ai/Trellis/marketplace -s trellis-meta` |
| [cc-codex-spec-bootstrap](/zh/skills-market/cc-codex-spec-bootstrap) | 用 CC + Codex 流水线引导生成编码规范 | `npx skills add mindfold-ai/Trellis/marketplace -s cc-codex-spec-bootstrap` |
| [frontend-fullchain-optimization](/zh/skills-market/frontend-fullchain-optimization) | 用 Web Vitals 诊断并优化前端性能 | `npx skills add mindfold-ai/Trellis/marketplace -s frontend-fullchain-optimization` |
