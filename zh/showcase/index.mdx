## 分享你的项目

用 Trellis 做了有意思的东西？我们想展示它。

我们在找什么：

- 真实项目（不是演示）
- 有意思地使用了 specs、tasks 或 multi-agent
- 愿意写一个简短介绍

要包含什么：

- 项目名称和描述
- 你怎么用 Trellis
- 什么效果好
- 你会做什么不同

通过 GitHub discussion 或邮件 taosu@mindfold.ai 提交。

---

## 展示标准

我们展示这样的项目：

**展示真实使用** - 不是只"装了 Trellis"，而是真正用它开发。

**能教东西** - 别人能从你的设置学到什么？

**有合理的文档** - 我们需要有东西可写。

我们不展示：

- 违反服务条款的项目
- 没有公开信息的项目
- 演示或玩具例子（除非有教育意义）
