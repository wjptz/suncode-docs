# 项目介绍

macOS 语音输入应用，用 Trellis 在 1 天内完成。

[![open-typeless](https://opengraph.githubassets.com/1/mindfold-ai/open-typeless)](https://github.com/mindfold-ai/open-typeless)

## 开发方式

从已有 Electron 项目复制 spec，AI 筛选整理后拆分成 3 个批次任务。当前 Trellis 推荐用原生 Git worktree + 每个 AI 窗口一个 session-scoped Trellis task 来做这类并行开发。
