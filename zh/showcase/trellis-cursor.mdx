# 项目介绍

社区 fork，专门为 Cursor 优化，支持中文子代理和 MCP 集成。基于 Trellis v0.2.12。

[![Trellis for Cursor](https://opengraph.githubassets.com/1/jojolionss/Trellis)](https://github.com/jojolionss/Trellis)

## 主要特性

- **Cursor Commands**: 13 个 slash commands，`.cursor/commands/` 格式
- **中文本地化**: 所有 commands 中文化
- **MCP 集成**: `trellis-context.*` 工具管理任务
- **多模型支持**: 可指定 `claude-4.5-opus-high-thinking` 等模型
