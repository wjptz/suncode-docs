## 主要内容

### 前端

- React 组件模式和 Hooks

- CSS 设计系统

### 后端

- API 模块结构

- 错误处理和日志

## 使用方法

1. 下载 ZIP 文件
2. 解压到项目的 `.trellis/spec/` 目录
3. 按你项目的具体规范修改
4. 删掉不适用的部分
5. 更新示例以匹配你的代码库
