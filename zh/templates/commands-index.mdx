Command 模板提供即用的 slash command，用于常见开发工作流。

## 可用模板

| 模板                               | 用途   | 说明         |
| ---------------------------------- | ------ | ------------ |
| [通用命令](/zh/templates/commands) | 工作流 | 常用开发命令 |

## 使用方法

1. 复制命令文件到 `.claude/commands/your-project/`
2. 按需重命名
3. 用 `/your-project:command-name` 测试
