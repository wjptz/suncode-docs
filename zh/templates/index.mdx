## 怎么用模板

### 规范

1. 选一个匹配你技术栈的模板
2. 复制到 `.trellis/spec/`
3. 按你项目的具体规范修改
4. 删掉不适用的部分

### 命令

1. 复制命令文件到 `.claude/commands/your-project/`
2. 按需重命名
3. 用 `/your-project:command-name` 测试

### 技能

1. 复制技能文件夹到 `.claude/skills/`
2. 技能在相关时自动加载
3. 有些技能需要配置 - 看 README

## 贡献模板

做了有用的东西？分享出来。

1. Fork Trellis 仓库
2. 把你的模板加到 `templates/`
3. 包含一个 README
4. 开 PR

好模板要具体、文档完善、在真实代码上测试过。
