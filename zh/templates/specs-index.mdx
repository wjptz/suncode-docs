Spec 模板帮你快速搭建项目的编码规范。复制、定制、使用。

## 可用模板

| 模板                             | 技术栈 | 说明         |
| -------------------------------- | ------ | ------------ |
| [通用 Spec](/zh/templates/specs) | 通用   | 通用编码规范 |

## 使用方法

1. 选一个匹配你技术栈的模板
2. 复制到 `.trellis/spec/`
3. 按你项目的具体规范修改
4. 删掉不适用的部分
