Spec 模板帮你快速搭建项目的编码规范。复制、定制、使用。

## 可用模板

| 模板                                                  | 技术栈 | 说明                 |
| ----------------------------------------------------- | ------ | -------------------- |
| [Node.js + Express](/zh/templates/specs-node)         | 后端   | Express + TypeScript |
| [Python + FastAPI](/zh/templates/specs-python)        | 后端   | FastAPI + SQLAlchemy |
| [React + TypeScript](/zh/templates/specs-react)       | 前端   | React + Zustand      |
| [Next.js + tRPC + Prisma](/zh/templates/specs-nextjs) | 全栈   | Next.js app router   |

## 使用方法

1. 选一个匹配你技术栈的模板
2. 复制到 `.trellis/spec/backend/` 或 `.trellis/spec/frontend/`
3. 按你项目的具体规范修改
4. 删掉不适用的部分
