把 `v0.5.11` 的 stable fix 同步到 0.6 beta 线。

## Bug Fixes

- **`add_session.py` / `task.py archive` 不再用 `git add -f` 强制 stage**。`.gitignore` 排除 `.trellis/` 时脚本打印 warning + 跳过 auto-commit。撤回 0.6.0-beta.5 里的 auto-retry。

## Enhancements

- **新配置 `session_auto_commit: true | false`** 在 `.trellis/config.yaml`（默认 `true`）。设 `false` 跳过 auto stage + commit；journal / archive 文件仍写入磁盘。老项目跑 `trellis update` 会通过 `configSectionsAdded` 自动追加注释掉的配置块。Closes [#245](https://github.com/mindfold-ai/Trellis/issues/245)。

## Upgrade

```bash
npm install -g @mindfoldhq/trellis@beta
trellis update
```
