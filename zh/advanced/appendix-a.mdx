## 附录 A：关键文件路径速查

<Note>
  本附录以 **Claude Code**
  的目录结构为例（`.claude/commands/`、`.claude/agents/`、`.claude/skills/`、`.claude/hooks/`）。其他平台的等价路径见下面「其他平台」小节，或[多平台章节](./multi-platform)的完整布局。
</Note>

### 核心文件

| 文件                                            | 说明                             | 读取时机                       |
| ----------------------------------------------- | -------------------------------- | ------------------------------ |
| `.suncode/workflow.md`                          | 开发工作流契约                   | 每次会话开始                   |
| `.suncode/config.yaml`                          | packages、update.skip、任务 hook | init 和 update                 |
| `.suncode/.version`                             | 当前 Suncode 版本                | update                         |
| `.suncode/.template-hashes.json`                | 模板文件 hash                    | update                         |
| `.suncode/.developer`                           | 开发者身份                       | 每次会话                       |
| `.suncode/.runtime/sessions/<session-key>.json` | 单个 AI 会话/窗口的 active task  | continue、hook、sub-agent 启动 |

### 原生命令（Claude Code）

| 文件                                      | 调用方式               |
| ----------------------------------------- | ---------------------- |
| `.claude/commands/suncode/finish-work.md` | `/suncode:finish-work` |
| `.claude/commands/suncode/continue.md`    | `/suncode:continue`    |

在 SessionStart 自动注入的平台上，`start` 不作为用户可见命令安装。agent-less 平台仍然保留手动 start workflow。

### 原生 sub-agent（Claude Code）

| 文件                                  | Sub-agent           | 角色                    |
| ------------------------------------- | ------------------- | ----------------------- |
| `.claude/agents/suncode-implement.md` | `suncode-implement` | 写代码，不 `git commit` |
| `.claude/agents/suncode-check.md`     | `suncode-check`     | 验证 + 自修复           |
| `.claude/agents/suncode-research.md`  | `suncode-research`  | 只读代码搜索            |

### 原生 skill（Claude Code）

| 文件                                          | Skill                 | 触发时机                  |
| --------------------------------------------- | --------------------- | ------------------------- |
| `.claude/skills/suncode-brainstorm/SKILL.md`  | `suncode-brainstorm`  | 用户描述功能 / bug / 请求 |
| `.claude/skills/suncode-before-dev/SKILL.md`  | `suncode-before-dev`  | 活跃任务中即将写代码      |
| `.claude/skills/suncode-check/SKILL.md`       | `suncode-check`       | 实现完成                  |
| `.claude/skills/suncode-update-spec/SKILL.md` | `suncode-update-spec` | 有值得沉淀的知识          |
| `.claude/skills/suncode-break-loop/SKILL.md`  | `suncode-break-loop`  | 刚修完棘手 bug            |

### Hook 脚本（Claude Code）

| 文件                                       | 触发               | 作用             |
| ------------------------------------------ | ------------------ | ---------------- |
| `.claude/hooks/session-start.py`           | SessionStart       | 自动注入上下文   |
| `.claude/hooks/inject-workflow-state.py`   | UserPromptSubmit   | 工作流状态面包屑 |
| `.claude/hooks/inject-subagent-context.py` | PreToolUse（Task） | 规范注入引擎     |
