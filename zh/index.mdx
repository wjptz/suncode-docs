## Trellis 是什么？

AI 的能力像藤蔓一样生长——充满活力但四处蔓延。Trellis 是 AI 的脚手架，引导它沿着规范的路径前进。

支持平台：Claude Code、Cursor、OpenCode、Codex、Kiro、Gemini CLI、Qoder、CodeBuddy、GitHub Copilot、Droid。

## 为什么用 Trellis？

| 功能             | 解决什么问题                                                                                       |
| ---------------- | -------------------------------------------------------------------------------------------------- |
| **自动注入**     | 规范和工作流自动注入每次对话，写一次，永久生效                                                     |
| **自更新规范库** | 最佳实践存在自更新的 spec 文件中，用得越多，AI 越懂你                                              |
| **Skill 优先**   | 大部分操作是 auto-trigger 的 skill —— AI 自己挑正确的那个，不用手动敲命令                          |
| **团队共享**     | 团队共享规范，团队里有一个高人搞一版好的规范，拉高全员的 AI coding 水平                            |
| **会话持久化**   | 工作记录持久化到仓库，AI 跨会话记住项目上下文，不用每次再费劲告诉 AI 你的项目情况是什么            |
